## PromptRails

Build, version, test, and observe AI agents and prompts. PromptRails gives you the infrastructure to go from prototype to production with confidence.

### SDKs

| Package | Version | Install |
|---------|---------|---------|
| [Python SDK](https://github.com/promptrails/python-sdk) | [![PyPI](https://img.shields.io/pypi/v/promptrails)](https://pypi.org/project/promptrails/) | `pip install promptrails` |
| [JavaScript SDK](https://github.com/promptrails/javascript-sdk) | [![npm](https://img.shields.io/npm/v/@promptrails/sdk)](https://www.npmjs.com/package/@promptrails/sdk) | `npm install @promptrails/sdk` |
| [Go SDK](https://github.com/promptrails/go-sdk) | [![Go](https://img.shields.io/github/v/tag/promptrails/go-sdk)](https://pkg.go.dev/github.com/promptrails/go-sdk) | `go get github.com/promptrails/go-sdk` |
